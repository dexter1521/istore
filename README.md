# iStore (Proyecto)

Este repositorio contiene la aplicacion iStore basada en Laravel.

## Estructura
- `src/src/` aplica Laravel (app real).
- `scripts/` utilidades locales.
- `project/` documentos internos (no se despliegan).

## Documentacion
- Lee `src/src/README.md` para setup y despliegue.

## Notas de despliegue
- Hosting compartido: apunta el document root a `src/src/public` o copia `public/` a la raiz del subdominio y ajusta `index.php`.
- PHP 8.2+ requerido.
